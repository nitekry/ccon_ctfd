# CTFd Terminal Theme

A dark, terminal/hacker-aesthetic theme for CTFd with green phosphor text, CRT scanline effects, and monospace typography.

## Features

- **Dark terminal aesthetic** вЂ” pitch black backgrounds with green (#00ff41) text
- **CRT scanline overlay** вЂ” subtle repeating gradient simulates old-school monitors
- **Monospace typography** вЂ” IBM Plex Mono + Share Tech Mono font pairing
- **Terminal UI language** вЂ” prompts (`>`), bracket-wrapped nav links, `root@ctf:~#` prefixes
- **Glowing text effects** вЂ” green text-shadow on headings and key elements
- **Full Bootstrap 5 override** вЂ” every component restyled (cards, modals, buttons, tables, forms, badges, pagination, etc.)
- **Boot-up flicker** вЂ” subtle opacity flicker on page load
- **Blinking cursor** вЂ” animation class for hero headings
- **Challenge board** вЂ” category filtering, modal challenge view, inline flag submission
- **All required CTFd templates** вЂ” login, register, challenges, scoreboard, settings, teams, users, and more

## Installation

### 1. Clone & Install Dependencies

```bash
git clone <this-repo> terminal-theme
cd terminal-theme
yarn install
```

### 2. Build Assets

```bash
# One-time build
yarn build

# Or watch mode during development
yarn dev
```

This compiles `assets/scss/main.scss` and `assets/js/index.js` into the `static/` directory and generates `manifest.json` + `manifest-css.json`.

### 3. Deploy to CTFd

**Self-hosted:**
Copy the theme folder into your CTFd installation:

```bash
cp -r terminal-theme /path/to/CTFd/CTFd/themes/terminal-theme
```

Then in the CTFd Admin Panel в†’ Settings в†’ Theme, select `terminal-theme`.

**Hosted CTFd (Professional/Enterprise):**
1. Remove `node_modules/` from the theme folder
2. Zip the theme: `zip -r terminal-theme.zip terminal-theme/ -x "terminal-theme/node_modules/*"`
3. Upload via Admin Panel в†’ Settings в†’ Theme

## Customization

All visual customization is in `assets/scss/main.scss`. Key variables to tweak:

```scss
$terminal-green:      #00ff41;   // Primary accent color
$terminal-bg:         #0d1117;   // Page background
$terminal-surface:    #131a24;   // Card/panel background
$terminal-border:     #1a2332;   // Border color
```

After editing SCSS, rebuild with `yarn build`.
